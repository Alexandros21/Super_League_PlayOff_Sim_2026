⚽ Playoff Title Simulator
A simple yet powerful web app that calculates championship chances based on all possible remaining scenarios.
This tool allows you to explore how the title race evolves by simulating every possible outcome of the remaining playoff matches.

🚀 Features
Scenario-based title chances
Calculates how often each team wins the title across all possible remaining match outcomes.
Interactive match input
Add results and scores to see live updates of the standings and title chances.
Advanced tie-break logic
Fully supports:
Double ties
Triple ties
Step-by-step tie resolution
Head-to-head analysis
Includes matches from:
Regular season
Playoffs
Provides a complete mini-league view for tie-breaking.
Detailed explanations
Understand why a team is ahead:
Which tie-break criterion is applied
What data is still missing
How ties are resolved step-by-step


🧠 How It Works
The app does not use predictive models or probabilities based on team strength.

Instead, it:
Takes all known results
Simulates all possible outcomes for remaining matches
Determines the champion in each scenario
Calculates how often each team finishes first

👉 This means the results represent:
“Scenario-based chances”, not real-world betting probabilities.
